##### IMDb Search - Консольное приложение для поиска фильмов

##### =======================================================

##### 

###### Описание

###### \--------

Программа для поиска фильмов и актеров из базы данных IMDb.

Позволяет искать фильмы по названию, смотреть детальную информацию

о фильме и актерах.



###### Файлы проекта

###### \-------------

IMDb.cpp       - главный файл, меню и интерфейс пользователя

functions.cpp  - функции для работы с данными

functions.h    - заголовочный файл

structs.h      - структуры данных



###### Структуры данных

###### \----------------

MovieShort     - краткая инфо о фильме (ID, название, год, рейтинг)

MovieDetails   - полная инфо о фильме (жанры, актеры, режиссеры, рейтинг)

ActorShort     - краткая инфо об актере в фильме

ActorDetails   - полная инфо об актере (годы жизни, профессии, известные работы)



###### Требования

###### \----------

\- Windows (UTF-8 кодировка)

\- Компилятор C++ (g++, MSVC, Clang)



Файлы данных (должны лежать в C:\\IMDb\_Data\_Analysis\\IMDb\\Temp\\):

1\. title.basics.tsv.txt     - основная инфо о фильмах

2\. title.ratings.tsv.txt    - рейтинги

3\. principals.tsv.txt       - съемочная группа

4\. name.basics.tsv.txt      - инфо о людях



Скачать можно с https://datasets.imdbws.com/



###### Поиск

###### \-----

\- Введи название фильма и нажми Enter

\- Напиши "exit" для выхода из программы



Навигация по списку фильмов:

\- p    - предыдущая страница

\- n    - следующая страница

\- 1-20 - выбрать фильм для просмотра

\- q    - новый поиск

\- Enter - обновить страницу



Просмотр фильма:

\- 1-N  - выбрать актера для просмотра

\- b    - вернуться к списку



Просмотр актера:

\- Enter - вернуться к фильму

Особенности

\-----------

\- Файлы читаются построчно (экономия памяти)

\- Формат TSV с разделителем табуляции

\- Обработка пустых значений (\\N)

\- Безопасное преобразование строк в числа

\- UTF-8 кодировка в консоли Windows
